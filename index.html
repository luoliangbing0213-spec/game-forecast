指标定义与计算公式 
【留存曲线】
  仅「次留（D1）」填绝对留存 %；D3、D7、D30、D60、D90 均由「相对次留」比例自动换算：
  R(Dn) = R(D1) × (Dn 相对次留 ÷ 100)。例：次留 50%，D1→3 相对次留 75% → 3日留存 = 50%×75% = 37.5%。
  关键帧：D0=100%，D1，D3，D7，D30，D60，D90。相邻帧之间在「对数空间」线性插值：
  ln R(d) = ln R(d_a) + (ln R(d_b) − ln R(d_a)) × (d − d_a) / (d_b − d_a)
  D90 之后沿用 D60→D90 的对数斜率外推；若相对比例导致节点非递减，会按 RPG 习惯压平为单调递减。

【DAU】
  设第 t 天新增登录为 N（当前模型为常数）， cohort 在满 d 天时的留存比例为 R(d)（D0 当天算 R(0)=1）。
  第 t 天 DAU(t) = Σ_{d=0}^{t−1} N × R(d)  （上线未满 t 天时按实际 cohort 天数截断）
  稳态近似：DAU ≈ N × Σ_{d=0}^∞ R(d)（用截断求和代替无穷）

【流水】
  日流水(t) = DAU(t) × ARPU

【实际收入】
  实际收入(t) = 流水(t) × (1 − 渠道分成)

【LTV（累计流水，未扣渠道）】
  以「单用户从注册日起」在窗口内的累计流水：
  LTV(Dn) = ARPU × Σ_{k=0}^{n−1} R(k)
  例如：LTV(D1) = ARPU×R(0)；LTV(D7) = ARPU×(R(0)+…+R(6))。

【按月统计表】
  日均新增 = 每日新增登录 N；月新增 = N × 当月天数。
  日均 DAU = (当月每日 DAU 之和) / (当月天数)。
  日均流水 = (当月每日流水之和) / (当月天数)；月流水 = 当月每日流水之和。
  月实际收入 = 当月每日实际收入之和（每日实际收入 = 当日流水 × (1−渠道分成)）。
  月营销费用 = 月新增 × 单用户成本；月毛利 = 月实际收入 − 月营销费用。

【可选：获客与 LTV 对比】
  单日新增总成本 = 每日新增登录 × 新增成本单价；可与 LTV×(1−渠道分成) 对比回收。
